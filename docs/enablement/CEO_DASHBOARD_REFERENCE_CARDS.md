## 📄 PRINTING INSTRUCTIONS

### Materials Needed:
- Heavy cardstock (110lb minimum)
- Color printer (or professional print shop)
- Laminator + 4×6" laminating pouches (or professional lamination)
- Paper cutter (for clean edges)

### Print Settings:
- **Paper Size:** 4×6" (postcard size)
- **Orientation:** Portrait
- **Print Quality:** High/Best
- **Color:** Full color
- **Duplex:** Double-sided (flip on short edge)

### Step-by-Step:
1. Print all cards on cardstock
2. Let ink dry completely (5-10 min)
3. Trim to exact 4×6" if needed
4. Laminate each card
5. Trim lamination edges
6. Distribute to operators

### Professional Alternative:
- Upload to Vistaprint/MOO/Staples
- Order as "4×6 Business Cards" or "Postcards"
- Select "Laminated" or "UV Coated" finish
- Order extras for new hires

### Storage & Distribution:
- Keep master copies in safe place
- Give each operator complete set (4 cards)
- New hires get set on Day 1
- Suggest laminating and keeping at desk
