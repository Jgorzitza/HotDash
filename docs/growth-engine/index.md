## Quick Reference

### Common Commands
```bash
# Initialize Growth Engine
npx tsx scripts/agent/get-my-tasks.ts <agent>

# Search KB
npx tsx scripts/agent/kb-search.ts <task-id> "<task-title>" <agent>

# Start task
npx tsx --env-file=.env scripts/agent/start-task.ts <TASK-ID>
```

### File Locations
- **Evidence Files**: `artifacts/<agent>/<date>/mcp/`
- **Heartbeat Files**: `artifacts/<agent>/<date>/heartbeat.ndjson`
- **Configuration**: `.env.local`
- **Services**: `app/services/`
- **Documentation**: `docs/growth-engine/`

### Key Services
- **MCP Evidence**: `app/services/mcp-evidence.server.ts`
- **Heartbeat**: `app/services/heartbeat.server.ts`
- **Dev MCP Ban**: `app/services/dev-mcp-ban.server.ts`

- **KB Integration**: `app/services/kb-integration.server.ts`
